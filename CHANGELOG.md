# 更新日志

## [1.1.0] - 2026-02-12

### 新增
- Send 类支持链式修饰方法：`At()`, `AtAll()`, `Reply()`
- 新增 `Raw_ob12()` 方法，支持发送 OneBot12 格式原始消息
- 方法名支持大小写不敏感调用（`Text`, `text`, `TEXT` 等效）
- 不支持的方法自动返回提示文本，便于调试

### 改进
- 完善消息发送 DSL 文档说明

## [1.0.0] - 2026-01-13

### 新增
- 初始版本发布
- 提供简洁的 MD3 风格网页界面
- 支持创建虚拟好友和群聊
- 通过 WebSocket 实时通信
- 支持发送和接收消息
- 消息记录功能
- 完整的 Send DSL 实现
- OneBot12 标准事件转换
